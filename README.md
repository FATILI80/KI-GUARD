# KI-GUARD

Rule-based early-warning filter for AI interaction drift.

Status: alpha research prototype.

Not a product.
Not a compliance system.
Not production-ready.
Not statistically or externally validated.

KI-GUARD is a deterministic, local, rule-based prototype for marking possible warning signs in AI-generated text.

It does not detect truth, intent, consciousness, deception, or actual manipulation.

Current runnable code:
ki_guard_v4_1.py

Architecture/specification document:
KI_GUARD_v5.0_ARCHITEKTUR.md

Important limits:
- mostly regex / marker based
- not robust against paraphrases
- can miss cases when wording changes
- can produce false positives
- does not replace human judgment
- not ISO-compliant
- not EU-AI-Act-compliant
- not production-ready
